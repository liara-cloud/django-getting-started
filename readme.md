# Django 5.2 + Object Storage S3

## Installation and Running

- Clone the repo to local:

```
git clone https://github.com/liara-cloud/django-getting-started.git
```

- Navigate to the needed dir:

```
cd django-getting-started
```

- Switch the branch to `object_storage`:
```
git checkout object_storage
```

- Install dependencies:
```
pip install -r requirements.txt
```

- Set ENVs in `settings.py`
- Run the project:
```
python manage.py runserver
```
